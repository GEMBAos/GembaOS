## The Auto-Kaizen Loop
To trigger the loop, ask your AI Copilot (me!) to "run the auto-kaizen workflow" or simply type `/auto-kaizen`.

The loop consists of:
1. **Analyze**: The agent reviews the current state of Kaizen Copilot.
2. **Brainstorm**: The agent identifies 1 high-impact feature or improvement to build.
3. **Plan**: The agent formulates an implementation strategy.
4. **Execute**: The agent writes the code and integrates the feature.
5. **Verify**: The agent tests that the application builds and the feature works.
6. **Log**: The agent documents the result in this file.
